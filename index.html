Setup

Link jQuery and file popup.functions.js in your project 

<script src="//ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js"></script> 
<script src="js/popup.functions.js"></script>

Standalone setup

Create a button 

<button class="standalone">Open filemanger</button>

Insert the launch code 

$(".standalone").click(function(){ popStandalone(); });

Input field setup

Create an input field 

<input type="text" id="file" name="file" >

Add the launch code 

$("#file").click(function(){ popFile("file"); });

cKeditor install

Install ckeditor http://ckeditor.com

Edit config.js in cKeditor and add: 

config.filebrowserBrowseUrl = 'path/to/esyFileManager/index.php?u=2';

tinyMCE install

Install tinyMCE http://www.tinymce.com

edit tinyMCE.init and add: 

file_browser_callback : "esyFileManage"

include tinyMCE_connect.js 

<script src="js/tinyMCE_connect.js"></script>

odify the path in tinyMCE_connect.js with esyfileManger path 

var cmsURL = "/esyFileManager/index.php?u=3"
